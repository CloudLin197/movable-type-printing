活字印刷云展馆 V2.47 修改说明

1. 课文知识小闯关固定按以下顺序显示：
   第1—5题：字词理解
   第6—7题：语句翻译
   第8—10题：内容理解
   各组内部随机抽题，选项仍随机打乱。

2. 解说词“历史评价”要求改为：
   联系课文开头的“雕版印刷”，说明从整版雕刻到单字组合的突破，并阐释它对印刷技术发展和知识传播的重要意义。

3. 梦溪支持：
   - 按住小人拖到页面任意任务位置；
   - 右上角减号隐藏；
   - 隐藏后通过页面右下角小头像恢复；
   - 拖动位置和隐藏状态自动保存；
   - 对话框会跟随梦溪的位置打开。

4. 探究对话记忆：
   - “活在哪里”完成思辨并点亮思辨之星后，再次提问会直接给出完整答案；
   - 其他引导题完成讨论后，再次提问会提示“刚刚我们已经讨论过这个问题啦”，并整理上次结论；
   - 讨论记录按当前学生分别保存。

覆盖项目中的同名文件后，提交到本地 Git，并在部署后使用强制刷新。
正式首页仅使用 index.html，不使用 index(12).html。
